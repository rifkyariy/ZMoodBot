## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/rifkyariy/ZMoodBot.git
    ```

2. Install Python and run: 
    ```sh
     pip3 install -r requirement.txt
    ```
